Book: Andrey Sadovykh (editor), Dragos Truscan (editor), Wissam Mallou - CyberSecurity in a DevOps Environment_ From Requirements to Monitoring (2023, Springer) - libgen.li
Chapter: 4 - “Toward Anomaly Detection Using Explainable AI.” This chapter presents 
MMT, a monitoring framework for anomaly detection. It is extended with explainable AI (XAI) capabilities for better understanding AI/ML-based clas-siﬁcations. The chapter includes experimental results using SHAP, LIME, and 
SHAPASH technologies.
